# HexaBot - Autonomous Cleaning Robot

## Overview
HexaBot is an intelligent autonomous cleaning robot designed using ESP32, ultrasonic sensors, IR sensors, and motor drivers for smart obstacle detection and navigation. The robot is capable of autonomous movement while performing floor cleaning tasks efficiently.

## Features
- Autonomous obstacle avoidance
- Ultrasonic and IR sensor integration
- ESP32 based control system
- Bluetooth control support
- Smart navigation system
- Cleaning mechanism integration

## Components Used
- ESP32
- Ultrasonic Sensor
- IR Sensors
- Motor Driver
- DC Motors
- Battery Module
- Chassis and Wheels

## Technologies Used
- Embedded Systems
- IoT
- Robotics
- Arduino IDE
- Sensor Integration

## Applications
- Smart home cleaning
- Robotics automation
- Industrial floor cleaning prototype
